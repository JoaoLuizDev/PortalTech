# Git Bash

Git Bash é um __shell (ambiente de linha de comando)__ que vem com o software Git para Windows. Ele __fornece uma interface de linha de comando__ para usar o Git e executar comandos em um repositório Git no Windows. O Git Bash é baseado no shell Bash do Linux e fornece uma experiência de linha de comando semelhante à do Linux no Windows.

O Git Bash permite que você execute os mesmos comandos Git que você executaria no Linux ou no Mac, incluindo clonar repositórios Git, criar ramificações, fundir ramificações, verificar o histórico de commits e muito mais. Ele também permite que você use alguns comandos do Linux no Windows, o que pode ser útil se você estiver acostumado a trabalhar em um ambiente de linha de comando do Linux.

Além disso, o Git Bash vem com ferramentas comuns do Linux, como o cat, grep e find, que podem ser usados para trabalhar com arquivos em um repositório Git. Em geral, o Git Bash é uma ferramenta útil para desenvolvedores que desejam usar o Git no Windows e preferem uma interface de linha de comando.
