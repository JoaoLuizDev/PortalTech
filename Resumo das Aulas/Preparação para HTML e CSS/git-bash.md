# Git Bash

Git Bash é um __shell (ambiente de linha de comando)__ que vem com o software Git para Windows. Ele __fornece uma interface de linha de comando__ para usar o Git e executar comandos em um repositório Git no Windows. O Git Bash é baseado no shell Bash do Linux e fornece uma experiência de linha de comando semelhante à do Linux no Windows.

O Git Bash permite que você execute os mesmos comandos Git que você executaria no Linux ou no Mac, incluindo clonar repositórios Git, criar ramificações, fundir ramificações, verificar o histórico de commits e muito mais. Ele também permite que você use alguns comandos do Linux no Windows, o que pode ser útil se você estiver acostumado a trabalhar em um ambiente de linha de comando do Linux.

Além disso, o Git Bash vem com ferramentas comuns do Linux, como o cat, grep e find, que podem ser usados para trabalhar com arquivos em um repositório Git. Em geral, o Git Bash é uma ferramenta útil para desenvolvedores que desejam usar o Git no Windows e preferem uma interface de linha de comando.

## Terminais

O terminal é um programa de computador que fornece uma interface de linha de comando para interagir com o sistema operacional. Ele permite que os usuários executem comandos digitados no teclado, em vez de clicar em botões ou ícones em uma interface gráfica do usuário.

Em um terminal, os comandos são digitados na linha de comando e executados pressionando a tecla Enter. O terminal exibe a saída dos comandos na tela, permitindo que os usuários visualizem e interajam com o sistema operacional de uma maneira mais poderosa e flexível do que em uma interface gráfica do usuário.

O terminal é uma ferramenta importante para desenvolvedores de software, administradores de sistemas e usuários avançados que precisam executar tarefas complexas em seus sistemas operacionais. Ele é amplamente utilizado em sistemas operacionais baseados em Unix, como Linux, macOS e FreeBSD, e também está disponível em outros sistemas operacionais, incluindo Windows.

Cada sistema operacional vem com um ou mais terminais instalados por padrão. No Windows, ele é o PowerShell.
