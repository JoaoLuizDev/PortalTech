| Comando       | Significado                     | Usado para                                                          |
| --------------| --------------------------------| --------------------------------------------------------------------|
| `pwd`         | Imprimir diretório de trabalho  | Mostrar o diretório atual em que você está trabalhando no terminal. |
| `cd`          | Mudar de diretório              | Navegar para um diretório diferente.                                |
| `cd ..`       | Mudar de diretório              | Navegar para o diretório pai do diretório atual.                    |
| `ls`          | Listar                          | Listar os arquivos e diretórios no diretório atual.                 |
| `mkdir`       | Criar diretório                 | Criar um novo diretório.                                            |
| `rmdir`       | Remover diretório               | Remover um diretório                                                |
| `code .`      | Visual Studio Code (VS Code)    | Abrir o editor de código VS Code no diretório atual.                |
| `git init`    | Git Initialize	              | Inicializar um repositório Git em um diretório local.               |
| `git clone`   | Git Clone 	                  | Clonar um repositório Git existente em um novo diretório local.     |
| `git add`	    | Git Add 	                      | Adicionar arquivos ao índice para serem commitados posteriormente.  |
| `git commit`  | Git Commit 	                  | Registrar as mudanças feitas nos arquivos adicionados ao índice.    |
| `git push`    | Git Push 	                      | Enviar as mudanças para um repositório remoto no GitHub.            |
| `git pull`    | Git Pull (Git puxar)	          | Mesclar mudanças mais recentes do rep. remoto com repositório local.|             
| `git status`  | Git Status 	                  | Verificar o estado atual do repositório Git.                        |
| `git branch`  | Git Branch (Git ramificar)	  | Listar, criar ou excluir ramificações (branches) do repositório Git.|
| `git checkout`| Git Checkout (Git verificar)	  | Alternar para uma ramificação específica do repositório Git.        |
| `git merge`   | Git Merge (Git mesclar)	      | Mesclar uma ramificação do repositório Git com outra.               |
